\doxysubsection{Descriere Detaliată}
Server API Flask pentru panoul de control și management al routerului virtual. 

Acest script expune endpoint-\/uri web destinate activării funcției de IP Forwarding, manipulării regulilor iptables pentru Firewall și Port Forwarding, precum și citirii logurilor generate de motorul din C.
